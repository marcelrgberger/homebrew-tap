# Homebrew Tap

Custom Homebrew tap for macOS apps by Marcel R. G. Berger.

## Usage

```bash
brew tap marcelrgberger/tap
```

## Available Apps

| App | Install | Description |
|-----|---------|-------------|
| [AutoBrew](https://github.com/marcelrgberger/auto-brew) | `brew install --cask autobrew` | Automatically keeps Homebrew packages up to date |

## Updating

```bash
brew update
brew upgrade --cask
```

Apps installed via this tap also auto-update via Sparkle.
