# Homebrew Tap

Custom Homebrew tap for macOS apps by Marcel R. G. Berger.

## Usage

```bash
brew tap marcelrgberger/tap
```

## Available

| App | Type | Install | Description |
|-----|------|---------|-------------|
| [AutoBrew](https://github.com/marcelrgberger/auto-brew) | Cask | `brew install --cask autobrew` | Automatically keeps Homebrew packages up to date |
| [AskaPro](https://github.com/marcelrgberger/askapro-cli) | Formula | `brew install askapro-cli` | AI-powered document analysis with 85+ expert consultation roles |

## Updating

```bash
brew update
brew upgrade
```
